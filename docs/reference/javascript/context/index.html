context

Functions exposed by the context API. context is global within your code. You do not have to do any imports.

Index​

Functions​

FunctionDescription
addCreate a dynamic property. TODO: remove first part of the path? TODO: create local?
getGets value of a property.
getExtendedGets extended value of a property. This is most useful for [html] slot properties.
mutateMutate value to of a property, best use with objects. You should use .mutate with objects because it prevents a new object from being created. If you call .set instead, you would need to provide a new object.
notifyProvide notification to convey status.
removeRemove a property. Only dynamically created properties can be removed. For static declared properties, be use hide. TODO: throw if attempt to remove static property
setWrite value to a property.
subscribeSubscribe to value changes to a single property or multiple properties.
subscribeAllSubscribe to all changes to any property.
triggerTrigger a trigger/event property.
